Readme

Features:

  • Add residue
  • Delete residue (right click on a node to open context menu). "Delete" will delete the monosaccharide and its children, "- Subs" deletes all the substituents attached to the node.
  • Select multiple nodes (hold Ctrl or Shift + left click).
  • Create Repeating Unit (In context menu, or press R). 4 boxes will appear to adjust the settings.
  • Export GlycoCT
  • Import GlycoCT
  • Quick Mode: Faster but only features the most common monosaccharides
  • Load structure
